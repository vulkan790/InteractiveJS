# InteractiveJS

Интерактивный тренажёр для практики навыков JavaScript. Веб - приложение с системой авторизации, задачами разных категорий и
уровней сложности, редактором кода с проверкой решений и статистикой процесса.

## Основные возможности

- **Регистрация и авторизация пользователей**
- **Каталог задач по 8 категориям JavaScript:**
  - Основы JavaScript
  - Условные операторы и циклы
  - Функции
  - Массивы и методы массивов
  - Объекты и ООП
  - DOM-манипуляции
  - Обработка событий
  - Асинхронный JavaScript
- **3 уровня сложности:** Легко, Средне, Сложно
- **Встренный редактор кода**
- **Автоматическая проверка решения через тесты**
- **Статистика процесса в профиле пользователя**
- **Фильтрация и поиск задач**
- **Адаптивный дизайн для всех устройств**

## Быстрый старт

### Предварительные требования
- Node.js (версия 16 или выше)
- npm или yarn

### Установка и запуск

1. **Клонируйте репозиторий**
```sh
git clone https://github.com/vulkan790/interactivejs.git
cd interactivejs
```

2. **Установите зависимости**
```sh
npm install
```

3. **Запустите в режиме разработки**
```sh
npm run dev
```

4. **Откройте в браузере**
http://localhost:5173

5. **Сборка для production**
```sh
npm run build
```

## Формат задач

Каждая задача хранится в taskData.js и имеет следующую структуру:
Пример задачи:

```js
{
    id: 'task-conditions-1',
    difficulty: 'medium',
    title: 'Проверка числа',
    description: 'Напишите программу, которая проверяет, является ли число положительным, отрицательным или нулём. Число задано в переменной num.',
    category: 'Условные операторы и циклы',
    categoryKey: 'Условные операторы и циклы',
    inputExample: 'let num = 5;',
    outputExample: 'Положительное',
    solution: 'let num = 5;\nif (num > 0) {\n  console.log("Положительное");\n} else if (num < 0) {\n  console.log("Отрицательное");\n} else {\n  console.log("Ноль");\n}',
    tests: [
        { input: 'let num = 5;', expected: 'Положительное' },
        { input: 'let num = -3;', expected: 'Отрицательное' },
        { input: 'let num = 0;', expected: 'Ноль' }
    ],
    starterCode: '// Пишите ваш код здесь\n'
}
```

## Хранение данных

В данный момент всё хранится локально, на браузере. В дальнейшем планируется интеграция с Backend
